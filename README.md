# Anggun Gadget Shop  
Full-Stack Rural E-Commerce System

Anggun Gadget Shop is a role-based, full-stack e-commerce web system developed as a Final Year Project (FYP) to digitalize a rural retail business in Sipitang, Sabah.  
The system replaces manual WhatsApp and walk-in order workflows with a centralized, database-driven platform.

It is designed with a modular backend architecture, secure authentication, and real-time order state management across Admin, Rider, and Customer roles.

## System Objectives
- Design a scalable web-based commerce system using PHP and MySQL  
- Implement role-based access control (RBAC) for Admin and Rider modules  
- Enable secure, cashless payment workflows  
- Provide verified customer feedback through an integrated rating system  

## Core Modules

Customer Module
- Product catalog & cart management  
- Order placement and tracking  
- Payment processing  
- Review & rating submission  

Admin Module
- CRUD operations for products and inventory  
- Order state management (Pending → Preparing → On Route → Delivered)  
- Delivery history and rider monitoring  

Rider Module
- Job assignment & acceptance  
- Delivery mode selection (direct / hub)  
- Proof-of-delivery upload  
- Earnings tracking  

## Tech Stack
PHP • MySQL • HTML • CSS • JavaScript  
Tailwind CSS • phpMyAdmin • XAMPP

## Architecture
- MVC-inspired folder structure  
- REST-style PHP endpoints  
- Centralized database schema  
- Session-based authentication  

## Impact
The system demonstrates how lightweight backend architectures can enable rural SMEs to adopt digital commerce with secure transactions, traceable deliveries, and scalable workflows.
