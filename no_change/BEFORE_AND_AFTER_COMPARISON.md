## Specific Examples

### Product Card Example

**BEFORE HTML:**
```html
<div class="box">
   <img src="product.jpg" alt="Product">
   <h3>Product Name</h3>
   <p class="price">$99.99</p>
   <button>Add to Cart</button>
</div>
```

**BEFORE CSS:**
```css
.box {
   background-color: #ffffff;
   box-shadow: 0 4px 8px rgba(0,0,0,0.1);
   border: 1px solid #cccccc;
   border-radius: 1rem;
}
```

**BEFORE APPEARANCE:**
Flat white card with generic gray border and heavy shadow

---

**AFTER HTML:**
```html
<!-- Same HTML - No changes needed! -->
<div class="box">
   <img src="product.jpg" alt="Product">
   <h3>Product Name</h3>
   <p class="price">$99.99</p>
   <button>Add to Cart</button>
</div>
```

**AFTER CSS:**
```css
.box {
   background: linear-gradient(180deg, rgba(255,255,255,0.98) 0%, rgba(248,249,250,0.8) 100%);
   box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08);
   border: 1px solid rgba(74, 144, 226, 0.06);
   border-radius: 1.2rem;
}
```

**AFTER APPEARANCE:**
Modern card with soft gradient, subtle brand border, refined shadow, and smoother corners
